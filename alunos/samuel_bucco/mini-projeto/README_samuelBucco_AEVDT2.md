## Como Executar

1. Abra a pasta `mini-projeto` no VS Code ou no Google Colab.
2. Abra o arquivo `mini_projeto_01.ipynb`.
3. Execute todas as celulas do notebook em ordem, de cima para baixo.
4. Verifique as analises, tabelas e conclusoes geradas ao final do notebook.
